Projekteinrichtung:
1) Installiere Unity in der Version 2020.3.12f1 LTS mit Android Support
2) Verbinde eine Oculus Quest + Controller
3) Öffne das Projekt
4) Öffne die Szene "MenuScene"
5) Drücke auf den Play Button

Steuerung:
Auswahlflächen über einen, von den Controllern ausgehenden, Laser Pointer highlighten 
und die Auswahl mit der Trigger-Taste bestätigen. 
Alles Weitere wird in der Anwendung erklärt.
